# ARPG Demo 项目规则

项目类型：Unity 2022.3 3D ARPG Demo。

目标：制作一个小型关卡制 ARPG Demo，包含主菜单、加载场景、战斗场景、角色移动、普攻、技能、怪物 AI、血条、技能 CD、伤害数字、结算界面。

代码要求：
1. 使用 C#。
2. 所有业务脚本放在 Assets/_Game/Scripts 下。
3. UI 使用 UGUI。
4. 配置优先使用 ScriptableObject。
5. UI 和战斗逻辑尽量通过事件解耦。
6. 高频对象如伤害数字、技能特效后续要接对象池。
7. 每次只实现一个小模块，不要大范围重构。
8. 代码要适合 Unity 初学者阅读，命名清晰，注释适量。
9. 不要引入大型第三方框架。
10. 不要改动无关文件。
